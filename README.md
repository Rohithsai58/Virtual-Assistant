# Virtual-Assistant
This project is a web-based Virtual Assistant built using HTML, CSS, and JavaScript. It is designed to interact with users through voice or text commands and perform basic tasks such as answering queries, opening websites, and providing real-time responses.

The assistant leverages JavaScript APIs like Speech Recognition and Speech Synthesis to understand user commands and respond in a natural, human-like manner. The user interface is simple, responsive, and visually appealing, ensuring a smooth user experience across devices.
